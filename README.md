# Cakra - Platform Edukasi Keuangan & Portofolio

Cakra adalah aplikasi edukasi berbasis web yang dirancang untuk membantu pengguna mempelajari literasi keuangan, menyelesaikan misi harian, mengikuti tantangan 15 hari, serta mensimulasikan pengelolaan portofolio investasi secara interaktif.

## 🚀 Fitur Utama

* **Auth & Placement Test:** Sistem masuk pengguna dilengkapi dengan uji penempatan (*placement test*) untuk menentukan tingkat kemahiran (Pemula, Menengah, Mahir).
* **Gamifikasi Sederhana:** Pengguna mendapatkan EXP setiap kali menyelesaikan modul atau misi untuk meningkatkan level.
* **Modul Edukasi:** Pembelajaran terstruktur berdasarkan tingkat kemahiran pengguna.
* **Asisten AI:** Konsultasi langsung mengenai topik keuangan menggunakan asisten berbasis kecerdasan buatan.
* **Misi & Tantangan 15 Hari:** Fitur pembiasaan harian untuk membangun konsistensi belajar.
* **Simulasi Portofolio:** Tempat pengguna mensimulasikan investasi secara aman tanpa risiko finansial nyata.

## 🛠️ Teknologi yang Digunakan

* **Framework:** Next.js (App Router, Client Component)
* **Bahasa:** TypeScript
* **Styling:** Tailwind CSS (dengan dukungan Dark Mode otomatis)
* **State Management:** React Hooks (`useState`, `useEffect`)
* **Penyimpanan Lokal:** LocalStorage (untuk fase pengembangan)
